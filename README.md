## About
Kotlin multiplatform project sharing code to Android, iOS and JS apps

<a name='Requirements'></a>

## Requirements
* Android Studio 4.2 or higher
* Java 8
* Gradle 6.7
* Kotlin 1.4.10
* Xcode 11+


<a name='Usage'></a>

## Usage
* Android: you'll need Android Studio 4.2 to run it.
* iOS: open iosApp.xcodeproj with Xcode 11+.
* JS: run `./gradlew :jsApp:browserDevelopmentRun`.
